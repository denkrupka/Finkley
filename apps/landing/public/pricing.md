# Pricing — Finkley

Finkley — управленческий учёт для салонов красоты. Цены окончательные, НДС не взимается (владелец освобождён от НДС). Валюта — EUR. Биллинг помесячный через Stripe. Отмена в один клик. Регион: Польша / ЕС.

Last updated: 2026-06-25

## Демо

- Price: €0 (14 дней)
- Card required: no
- Limits: весь функционал без ограничений на 14 дней
- Features: все разделы и интеграции, без лимитов, без привязки карты. После — выбор любого тарифа.

## Бесплатный (Free)

- Price: €0/month (навсегда)
- Card required: no
- Limits: открыт только раздел «Доходы» (визиты и выручка); остальные разделы видны, но заблокированы
- Features: учёт визитов и выручки, без срока действия, апгрейд в один клик

## Старт (Start)

- Price: €19/month
- Features: всё из бесплатного + раздел «Расходы» + раздел «Отчёты» + раздел «Мессенджер»
- Use case: полный учёт денег — доходы, расходы и отчётность

## Рост (Growth) — самый популярный

- Price: €49/month
- Features: всё из €19 + раздел «Маркетинг» + AI-помощник
- Use case: учёт плюс привлечение клиентов и AI-аналитика

## Полный (Full)

- Price: €69/month
- Features: всё из €49 + раздел «Финансы» (P&L и движение денег / ДДС) + раздел «Склад» + все интеграции
- Use case: все разделы продукта

## Сеть (Network / Multi-salon)

- Price: €99/month
- Features: всё из €69 + несколько салонов на одном аккаунте + переключатель салонов + каждый салон считается отдельно
- Use case: владелец нескольких салонов

## Notes

- VAT: не взимается, в чеке ровно сумма тарифа
- Payment methods: Visa, Mastercard (через Stripe)
- Cancellation: в один клик в Настройках → Подписка; до конца оплаченного периода тариф активен, затем переход на бесплатный тариф
- Refunds: возврат в первые 14 дней оплаченной подписки — написать на info@finkley.app
- Data export: CSV в любой момент, без привязки к сервису
- Полная страница тарифов: https://finkley.app/pricing
