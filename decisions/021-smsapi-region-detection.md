# ADR-021: SMSAPI endpoint — .pl по умолчанию

**Дата:** 2026-05-22
**Статус:** Принято
**Контекст:** Коммит `16aba95`

## Контекст

SMSAPI (smsapi.com / smsapi.pl) предоставляет 2 независимых
региональных endpoint'а:

- `api.smsapi.com` — международный (RU/UA/etc)
- `api.smsapi.pl` — польский

OAuth-токены **не shared между регионами**: токен issued на
`ssl.smsapi.pl/login` НЕ работает на `api.smsapi.com` (HTTP 401
`authorization_failed`). И наоборот.

## Проблема

Изначально (коммит `230ef7e`) код хардкодил `api.smsapi.com` в обоих
helper'ах (`_shared/sms.ts`, `_shared/smsapi-sender.ts`).

При первом реальном тесте после деплоя в проде юзер получил 401 от
SMSAPI. Диагностика заняла ~30 мин: думали что токен просроченный,
советовали юзеру сгенерировать новый. Только после прямого `curl`
к обоим endpoint'ам выяснилось — токен валидный, но на `.pl`.

## Решение

Дефолтный endpoint = `api.smsapi.pl`, с override через ENV:

```ts
const base =
  Deno.env.get('SMS_API_REGION') === 'com' ? 'https://api.smsapi.com' : 'https://api.smsapi.pl'
```

Обоснование выбора `.pl` как дефолт:

- Основной рынок FinSalon = Польша (валюта PLN, локаль PL, KSeF/wFirma
  интеграции для PL).
- На SMSAPI польские регистрации чаще, чем международные (исторически
  компания SMSAPI родом из Польши).
- Override через ENV — простой эскейп-хач для не-PL юзеров.

## Последствия

- Польские юзеры — Just Works без настройки.
- Международные юзеры (если появятся): ставят `SMS_API_REGION=com`
  в Supabase Function Secrets, и всё работает.
- Документация: добавлено в комментарии `_shared/sms.ts` (читается
  каждым кто туда заглянет).

## Альтернативы (отвергнутые)

1. **Auto-detect**: пробовать оба endpoint'а при первом запросе, кешировать.
   Минус: лишний раунд-trip + сложность кеша между cold-start'ами Deno.
2. **Хранить region per salon в БД**: излишне для текущего масштаба
   (1 SMSAPI аккаунт на весь FinSalon).
3. **Спросить юзера при подключении**: добавляет шаг в UI ради edge case.

## Что нужно сделать дальше

- При запуске международной экспансии — UI-кнопка в Settings →
  Интеграции → SMS «Я не в PL» → выставляет ENV для салона (или
  per-salon override через JSON).
- Тесты на endpoint detection — пока не критично, behavior статичный.
